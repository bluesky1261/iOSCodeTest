# iOSCodeTest
코드 테스트

## Project Description
* Unsplash 메인 / 검색 화면 구현
* Viper Architecture를 사용하여 구조화
* Image 조회 성능 개선을 위하여 Screen Size에 맞는 이미지 조회 + 이미지 Caching 구현

### 추가 구현 기능
* 검색어 History 기능
* Category별 검색 기능
